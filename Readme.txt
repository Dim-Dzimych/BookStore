ТЗ выполнено согласно полученным требования.

1) Для включения и проверки у себя требуется сменить ConnectionString в appsetting.json

2) При желании измненить таблици сущностей после добавления/изменения крайних, требуется выполнить миграцию.Для этого откройте Package Manager Console выберите Bookstore.Domain и вставьте следующую команду 
- Add-migration "<Migration name>" | После этого Вам потребуется обновиться Базу Данных
- Update-database | Теперь наслаждаемся.
